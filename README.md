# 🌱 Kisan Mitra

AI-powered Smart Farming Android Application built using Kotlin, Jetpack Compose, Firebase, Gemini AI, and TensorFlow Lite.

---

## 📱 About the App

Kisan Mitra is a smart farming assistant application designed to help farmers with crop disease detection, multilingual AI guidance, weather updates, and farming tools.

The app allows users to scan crop images using AI and receive disease predictions along with treatment and prevention recommendations.

---

## ✨ Features

- 🌾 AI Crop Disease Detection
- 🤖 AI-powered Farming Assistant
- 🌍 Multilingual AI Support
- ☁️ Weather Updates
- 🧪 Fertilizer Calculator
- 🛡️ Pesticide Recommendation System
- 👤 Detection History Tracking
- 📚 Agriculture Learning Section

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Firebase
- Gemini AI API
- TensorFlow Lite (.tflite)
- REST APIs
- Material 3 UI
